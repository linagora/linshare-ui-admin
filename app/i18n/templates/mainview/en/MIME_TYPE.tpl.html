Gestion des règles de type MIME

Cette vue vous permet de paramétrer les filtres par type MIME. Ainsi vous pouvez autoriser ou interdire le dépôt de fichiers d’un type donné, e.g. interdire le dépôt de fichiers vidéos (MPEG, etc.).

Fonctionnement :

  • Sélectionnez un domaine depuis l’« Arbre des domaines », cliquez sur le bouton « Ajouter » et créer votre filtre;
  • Cliquez ensuite sur le filtre de votre choix ;
  • Pour chaque type MIME, activez-le ou désactivez-le, puis enregistrez en cliquant sur le bouton « Sauvegarder ».
